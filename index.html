1. html

Início

É utilizada para informar ao navegador
que tipo de estrutura é essa que está sendo
construída, podendo ser títulos, parágrafos,
imagens, links, entre outros..

2. head

Início

É usado para informações gerais (metadados)
sobre o documento, incluindo seu título e links
para scripts e folhas de estilos.

3. body

Início

É onde colocamos todos os elementos visíveis
de uma pagina, por exemplo tudo que você
vê nessa página está dentro do elemento body.

4. Tags para títulos
(h1, h2, h3, h4, h5 e h6)

Início

Usamos para criar títulos em páginas web.

O significado dessas tags é o mesmo, que é definir títulos,

mas o nível de importância é diferente para cada uma,

onde h1 tem o nível mais alto e h6 o mais baixo.

5. input

Início

6. strong

Início

Colocamos apenas para ênfatizar uma escrita
no texto deixando em negrito

7. i

Início

Colocamos para representar uma parte do texto
com uma diferença semântica, na qual a representação
tipográfica padrão é no tipo itálico. Isso significa que os
navegadores continuarão a apresentar o conteúdo em itálico,
mas, de acordo com o que é definido, isso não é mais requerido.

8. blockquote

Início

Colocamos para indicar que o texto dentro da tag

é uma pequena citação.
Este elemento destina-se a citações curtas
que não requerem marcações de parágrafo.

9. a

Início

É usada para criar uma âncora de onde vincular,
o atributo href é usado para onde endereçar o documento,
e as palavras entre as tags de abertura e fechamento
da âncora serão exibidas como um hipervínculo.

10. ul

Início

  • Usamos para elaborar uma lista de itens sem ordem rígida,
  • isto é, uma coleção de itens que não trazem uma ordenação
  • numérica e as suas posições, nessa lista, são irrelevantes.

11. ol

Início

  1. Representa uma lista de itens ordenados.
  2. De forma característica esses itens ordenados
  3. em uma lista são mostrados com uma contagem que os precede,
  4. que pode ser de qualquer tipo, como numerais,
  5. letras, algarismos romanos, ou simples símbolos.

12. li

Início

É usado para representar um item que faz parte
de uma lista. Este item deve estar contido em
um elemento pai: uma lista ordenada (ol),
uma lista desordenada (ul) , ou um menu (menu)
